Tentang Proyek RecurCrawl

RecurCrawl adalah aplikasi web yang dirancang untuk menganalisis dan membandingkan efisiensi dua pendekatan algoritma fundamental—iteratif dan rekursif—dalam konteks web crawling data dari platform media sosial Threads.

Analisis Performa

Membandingkan waktu eksekusi (runtime) antara metode iteratif dan rekursif pada berbagai ukuran dataset.

Fungsi Utama

Mengumpulkan data balasan dari platform Threads untuk menguji performa algoritma pada dataset nyata.

Teknologi

Dibangun menggunakan Python dengan framework Flask untuk backend, serta HTML dan Tailwind CSS untuk frontend.

Bagaimana Cara Kerjanya?

1

Unggah Dataset

Pilih dan unggah berkas JSON yang berisi data dari Threads.

2

Tentukan Ukuran

Masukkan ukuran dataset yang ingin diuji, dipisahkan dengan koma.

3

Jalankan Analisis

Klik tombol "Jalankan Analisis" untuk memulai proses perbandingan.

4

Lihat Hasil

Hasil perbandingan akan ditampilkan dalam bentuk grafik dan tabel.
